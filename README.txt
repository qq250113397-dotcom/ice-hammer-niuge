雷姆 自定义宠物安装说明

1. 解压这个文件夹。
2. 把整个文件夹放到你的 Codex Home 下面的 pets 目录里。
3. 最终路径应像这样：
   - macOS / Linux: ~/.codex/pets/ice-hammer-niuge/
   - Windows: %USERPROFILE%\\.codex\\pets\\ice-hammer-niuge\\
4. 打开 Codex，进入 Settings > Appearance > Pets。
5. 刷新自定义宠物后，选择 雷姆（或 `ice-hammer-niuge`）。

文件说明：
- pet.json: 宠物配置
- spritesheet.webp: 宠物动画图集

互动说明：
- 打招呼：看起来像主动挥手问好
- 挥动流星锤：作为工作/行动状态的武器动作
- 施展冰魔法：单独出现，不和流星锤同屏
